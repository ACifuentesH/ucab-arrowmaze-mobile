# UCAB Arrow Maze — Mobile Client

Flutter client for **Arrow Maze — Escape Puzzle** (Software Development semester project).

- Architecture: **MVVM + Clean Architecture** (`domain` / `application` / `infrastructure` / `presentation` / `config`).
- Backend: [`ucab-arrowmaze-api`](docs/backend-context.md) (REST, JWT).
- Testing methodology: [testing-architecture.md](docs/testing-architecture.md) (Test / Testing API / Object Mother, AAA).

> Full README (badges, screenshots, SOLID/patterns/AOP sections) is completed at the docs phase.

## Getting Started

```bash
flutter pub get
flutter run
```

## Running Tests

```bash
flutter analyze
flutter test
```
